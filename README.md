# Decodelabs-Internship-Projects

# 📌 Project Overview

This project focuses on analyzing an e-commerce sales dataset to uncover valuable business insights regarding sales performance, customer purchasing behavior, product performance, payment preferences, referral sources, and order fulfillment trends.

The project followed a complete analytics workflow:

1. Data Cleaning using Microsoft Excel
2. Exploratory Data Analysis (EDA) using Excel Pivot Tables
3. Business Analysis using SQL Queries
4. Dashboard Development using Power BI

# 🛠️ Tools and Technologies

| Tool | Purpose |
|--------|----------|
| Microsoft Excel | Data Cleaning and Exploratory Data Analysis |
| SQL | Business Query Analysis |
| Power BI | Dashboard Creation and Data Visualization |
| Pivot Tables | Aggregation and Trend Analysis |
| DAX | KPI Calculations in Power BI |


# 📂 Dataset Overview

The dataset contains **1,200 e-commerce transactions** collected between 2023 and 2025.

### Dataset Features

# 🧹 Data Cleaning Procedures

The dataset was cleaned using Microsoft Excel to ensure accuracy and consistency before analysis.

### Cleaning Steps Performed

- Checked for missing values
- Removed duplicate records
- Standardized column names
- Verified data types for all columns
- Validated numerical calculations
- Ensured date consistency
- Checked for outliers and anomalies
- Formatted fields appropriately

### Outcome

A clean and structured dataset ready for exploratory analysis, SQL querying, and dashboard development.
